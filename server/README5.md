# Realtime Chat (Socket.io + React)

Features implemented:
- Global chat room
- Username-based login (stored locally)
- Private messaging (pm:room pattern)
- Typing indicators
- Online/offline status
- File sharing (base64, downloads)
- Read receipts (explicit mark-read + propagation)
- Pagination (load older messages)
- Reconnection logic (socket.io built-in)
- Browser notifications and sound (basic)
